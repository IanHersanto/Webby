Webby HTTPd installation wizard for linux

Automate your web server installation on Ubuntu, Debian, Fedora, CentOS, or RHEL

About

What is Webby? Webby is a Free utility licensed under MIT License to help you install and configure your linux server including web server software, database software, PHP/Perl scripting, FTP server, and also Firewall. Maybe you're a web programmer who need to prepare your own development server on your own linux box, but you don't know how to configure your linux server. Some people may experience hours to a day to get reference in configuring linux server for spesific distribution, you may experience compiling errors on some software with different libraries. With webby you can install your prefered software as you like, without worrying dependency errors for each software libraries, and what you've to do is just waiting like you do this stuff in windows. Webby will give you a wide range of software to install. Here are the list of software to install :
